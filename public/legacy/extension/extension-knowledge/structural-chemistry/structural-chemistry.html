分子轨道理论：从杂化到统一图景

写在前面

本节衔接高中杂化轨道理论，引入 分子轨道（MO）理论。杂化理论擅长预测构型，MO 理论则给出更深的量子图像，解释电子结构与性质。掌握 MO 理论后，可统一理解 H2、O2、苯以及金属固体等键合现象。

杂化轨道：优秀的几何预测工具

杂化轨道将原子轨道（s、p）线性组合为 sp3、sp2、sp 等新轨道形成 σ 键，解释 CH4（正四面体）、C2H4（平面三角）、C2H2（直线形）等构型。

但杂化理论在以下现象面前显得无力：

O2 顺磁性、H2+ 的稳定性、苯与 1,3-丁二烯的真实离域，难以用简单的杂化和共振清楚解释。

MO 理论：电子离域到整个分子

在 MO 理论中，电子不再局限在某个键上，而是分布在整个分子空间。分子轨道由 原子轨道线性组合（LCAO） 形成，类似声波叠加成新的复合波。

电子云密度与 |ψ|² 成正比，因此轨道重叠会直接改变电子最可能分布的位置。

同相与反相叠加

两个原子轨道（ψA 和 ψB）可写为 ψ MO = cA ψA ± cB ψB。

同相叠加在核间增强电子密度，能量降低，形成 成键轨道（σ、π）。反相叠加在核间形成节面，能量升高，形成 反键轨道（σ*、π*）。有些轨道几乎不组合，称为 非键轨道。

能级图与键级

分子轨道按能量高低排列形成能级图，电子遵循泡利原理、能量最低原理与洪特规则填入。键级为：

键级 = （成键轨道电子数 - 反键轨道电子数）/ 2。

经典例子：H2、He2、O2

H2：两个电子填入 σ 1s 成键轨道，键级 1，形成稳定单键。

He2：四个电子填满 σ 1s 与 σ 1s*，键级 0，不稳定。

O2：同核双原子分子中（考虑 s-p 混杂），价层能级顺序为 σ2s < σ2s* < σ2pz < π2px = π2py < π2px* = π2py* < σ2pz*。最后两个电子占据简并 π* 轨道且自旋平行，解释 顺磁性，键级为 2。

威力与统一性

MO 理论解释共轭与芳香性：苯中 π 电子离域形成稳定闭壳层，得到 4n+2 规则。

将 MO 思想推广到固体得到 能带理论（价带与导带），电子填充决定导体、半导体或绝缘体。

小结

杂化轨道是几何预测与局部 σ 键的优秀工具；MO 理论是解释磁性、光谱、电离能、电子亲和能与离域的更深框架。二者互补而非互斥。

类比：杂化理论像城市道路图（原子如何连接），MO 理论像电力与通信网络（电子如何运动）。

从定域“键”到离域“分子轨道”，是化学由经典图像迈向量子图像的重要一步。

晶体场理论：d 轨道分裂与构效关系

写在前面

配合物是无机化学核心。晶体场理论（CFT）聚焦 d 轨道分裂，解释配合物结构、颜色、磁性与稳定性。本节不引入 MO 理论，仅在 CFT 框架内讨论。

基础要点回顾

配位键本质：CFT 将配位键视作中心金属阳离子与配体点电荷/偶极子的静电吸引。

配位数与构型：六配位常为正八面体（如 [Co(NH3)6]3+、[Fe(H2O)6]2+）；四配位常为正四面体（如 [ZnCl4]2-）或平面正方形（如 [Ni(CN)4]2-）。

d 轨道形状：dz2 沿 z 轴伸展并在 xy 平面形成环；dx2-y2 指向 x、y 轴；dxy、dxz、dyz 位于轴间并沿 45 度方向。

八面体场分裂

球形场中五个 d 轨道简并；八面体场中，指向配体的 eg（dz2、dx2-y2）能量升高，指向轴间的 t2g（dxy、dxz、dyz）能量较低。

分裂能为 Δo（亦写 10Dq）。能量守恒给出 eg 为 +0.6 Δo、t2g 为 -0.4 Δo；用 Dq 表示即 eg 为 +6Dq、t2g 为 -4Dq。

影响 Δ 的因素

配体场强度：弱到强的光谱化学序列：I- < Br- < Cl- < SCN- < F- < OH- < H2O < SCN-(S) < NH3 < en < bipy < phen < NO2- < CN- ≈ CO。强场配体 Δ 更大。

口诀："碘溴氯硫氰氟，氢氧水硫氰氨，乙二胺联吡邻菲啰啉，硝基氰碳氧终最强"。

金属因素：氧化态越高、周期越大，Δ 越大。

应用：颜色、磁性、稳定性

颜色：d-d 跃迁吸收能量等于 Δ。Δ 越大吸收越高能，颜色随之变化。[Cu(H2O)6]2+ 为淡蓝色，[Cu(NH3)4(H2O)2]2+ 为深蓝色。

高低自旋：比较 Δ 与成对能 P。Δ < P 为高自旋；Δ > P 为低自旋。例：d6 Fe2+ 在 [Fe(H2O)6]2+ 中为高自旋，在 [Fe(CN)6]4- 中为低自旋（t2g6 eg0）。

CFSE：晶体场稳定化能 CFSE = (-0.4 n t2g + 0.6 n eg) Δo，影响水合焓、晶格能、稳定性与构型选择（含 Jahn-Teller 畸变）。

其他配位场

正四面体：分裂顺序反转（e 低、t2 高），Δt 较小（约为 Δo 的 4/9），几乎总为高自旋。

平面正方形：八面体去掉 z 轴配体的极端畸变。dx2-y2 最高，dxy 次之，dz2 再次，dxz/dyz 最低。强场配体促使 d8 离子形成平面正方形。

小结

CFT 用 d 轨道分裂与 Δ 构建统一框架，解释配合物的结构、颜色、磁性与稳定性，是高中配位化学通向更深无机化学的关键桥梁。
