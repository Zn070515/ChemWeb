电极电势：核心概念

写在前面

本节以电化学为切入点拓展氧化还原概念，重点解释电极电势及其热力学意义。原文作者提示自己可能久未翻阅教材，建议结合课本核对。

电势差从何而来

在 Zn-Cu 原电池中（CuSO4 电解质），Zn 更易失电子，Cu2+ 更易得电子沉积。Zn 电极失电子后带负电，附近溶液带正电，从而形成 电极电势。

电极电势不仅存在于金属，任何能发生氧化还原的固体、液体、气体都可能具有电极电势。

标准氢电极（SHE）

实验只能测电势差，因此规定标准氢电极（100 kPa H2、1 mol/L H+、铂黑电极）电势为 0.00 V。与其组成原电池测得的电势差即该电极电势。

标准条件下（室温、1 mol/L、100 kPa）测得的电势称 标准电极电势 Eθ。例：Eθ(Zn2+/Zn) = -0.76 V，Eθ(Cu2+/Cu) = +0.34 V，电池电压为 1.10 V。

该例假设有盐桥连接 1 mol/L 的 ZnSO4 与 CuSO4 溶液。

记法与正负

Ox + n e- -> Red，记作 Eθ(Ox/Red)。例如：

Zn2+ + 2 e- -> Zn，Eθ(Zn2+/Zn) = -0.76 V。

同一对电极的氧化/还原电势数值相等、符号相反，很多表格默认给出氧化电势。

Eθ 越正氧化性越强，越负还原性越强。

热力学联系：ΔG 与能斯特方程

ΔG 与 Etheta

电池的最大非体积功等于电功。W = Q E，Q = nF，则：

ΔGθ = -nF Eθ。

又有 ΔGθ = -RT ln K，因此：

Eθ = (RT / nF) ln K。

F 为法拉第常数（1 mol 电子电量，F = e NA）。

能斯特方程

非标准条件下：Δ G = ΔGθ + RT ln Q，得到：

E = Eθ - (RT / nF) ln Q；25℃ 时 E = Eθ - (0.0592 / n) lgQ。

例：Zn2+ = 0.01 mol/L 时，E(Zn2+/Zn) = -0.76 - 0.0592/2 * lg(0.01) = -0.708 V。

应用：氯碱工业

总反应：2 NaCl + 2 H2O（电解）-> 2 NaOH + H2 + Cl2。

Cl2 与 NaOH 同处一室会歧化：Cl2 + 2 OH- -> Cl- + ClO- + H2O，因此需用离子交换膜隔开。

电极反应

阴极：2 H2O + 2 e- -> 2 OH- + H2。

阳极：2 Cl- -> Cl2 + 2 e-。

放电顺序定量分析

298 K，0.1 mol/L NaCl：

Na+ + e- -> Na，E = -2.71 + 0.0592 lg 0.1 = -2.76 V。

Cl2 + 2 e- -> 2 Cl-，E = 1.36 - 0.0592/2 * lg 0.12 = +1.42 V。

2 H+ + 2 e- -> H2，E = 0 + 0.0592/2 * lg(10-7)2 = -0.4144 V。

O2 + 2 H2O + 4 e- -> 4 OH-，E = 0.401 - 0.0592/4 * lg(10-7)4 = +0.8154 V。

阳极为氧化过程，电势取反，可判断阳极放电为 Cl-，阴极为 H2。

热力学代价

ΔG = 2 Δf Gm(OH-) + Δf Gm(H2) + Δf Gm(Cl2) - 2 Δf Gm(H2O) - 2 Δf Gm(Cl-)。

代入数值得 ΔG = 422.2 kJ/mol，反应非自发，需输入电能，但产物价值高，工业上可行。

提示：ΔG = ΔH - T ΔS，ΔH 为反应焓变，ΔS 为熵变。

附录提示

标准电极电势、酸表、碱表常作为附录表格，便于快速解题。
