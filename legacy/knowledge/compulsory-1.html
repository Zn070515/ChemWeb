第一章：物质及其变化
1.1 物质的分类及其转化
  1. 物质的分类：纯净物与混合物

    是否由多种物质组成来区分纯净物与混合物；纯净物包括单质和化合物，混合物可分为均匀混合物（溶液）和非均匀混合物（胶体和悬浊液）。 单质是由同一种元素组成的纯净物，化合物是由两种或两种以上元素按一定比例组成的纯净物。 比如O2和O3混合，就是混合物；NaCl是化合物，Cu是单质。 此外物质分类有树状分类法和交叉分类法两种常见方式。

  2. 分散系的类型：溶液、胶体与悬浊液

    根据分散相粒径大小，分散系可分为溶液（粒径<1nm）、胶体（1nm-100nm）和悬浊液（>100nm）。 溶液是均匀混合物，胶体和悬浊液是非均匀混合物。 胶体具有丁达尔效应，而溶液不具有；悬浊液中的颗粒较大，易沉降。

  3. 胶体的丁达尔效应与聚沉

    丁达尔效应是胶体分散系中悬浮颗粒对光线的散射现象，表现为光路可见。 聚沉是指胶体颗粒在电解质作用下聚集沉降的过程，可通过加入电解质或改变pH值实现。 电泳现象是胶体颗粒在电场作用下向电极移动的现象，反映了胶体颗粒的带电性质。

  4. 物质的转化

    物质可以通过物理变化（如溶解、混合、过滤等）和化学变化（如反应生成新物质）进行转化。 物理变化不改变物质的化学性质，化学变化则涉及化学键的断裂和形成。 例如，盐溶于水是物理变化，而铁生锈是化学变化。 物质的转化可用于制取某种物质，这时要考虑到各物质的性质。如果是工业生产，还要考虑到成本、安全性、环保等因素。

1.2 离子反应
  1. 离子反应的概念与特点

    离子反应是指溶液中离子之间发生的化学反应，通常涉及离子的结合、置换或沉淀形成。 离子反应的特点包括反应速度快、易于观察和测量，以及通常在水溶液中进行。 例如，NaCl溶液与AgNO3溶液反应生成AgCl沉淀，即为离子反应。

  2. 什么是电解质

    电解质是指在水溶液或熔融状态下能够导电的物质，因其分子在溶液中能够解离成带电的离子。 根据解离程度，电解质可分为强电解质（如NaCl、HCl）和弱电解质（如CH3COOH、NH3）。 强电解质在水中几乎完全解离，而弱电解质只部分解离。

  3. 离子方程式的书写方法

    离子方程式表示溶液中实际发生的离子反应过程，书写时需将强电解质完全离解为离子形式，弱电解质和沉淀以分子形式表示。 步骤包括：写出完整的化学方程式，分解强电解质为离子形式，取消观众离子，最后整理成离子方程式。 例如，AgNO3与NaCl反应的离子方程式为：Ag⁺(aq) + Cl⁻(aq) → AgCl(s)。 常见的不能拆的物质包括水、沉淀、弱电解质（如醋酸、一水合氨等）和气体。还有浓硫酸这种较为特殊的例子。

  4. 常见离子反应类型

    常见的离子反应类型包括沉淀反应、酸碱中和反应和氧化还原反应。 沉淀反应是指溶液中生成不溶性固体（沉淀）的反应，如AgNO3与NaCl生成AgCl沉淀。 酸碱中和反应是指酸与碱反应生成盐和水的过程，如HCl与NaOH反应生成NaCl和H2O。 氧化还原反应涉及电子转移，如Zn与Cu²⁺反应生成Zn²⁺和Cu。

  5. 沉淀、气体与弱电解质生成条件

    沉淀生成的条件通常是两种可溶性盐溶液混合后形成不溶性盐，如BaCl2与Na2SO4反应生成BaSO4沉淀。 气体生成的条件包括酸与碳酸盐反应生成CO2气体，或金属与酸反应生成H2气体。 弱电解质的生成通常涉及酸碱中和反应，如HCl与NH3反应生成NH4Cl弱电解质。

  6. 离子共存与反应判断

    判断离子是否能共存主要依据溶解度规则和反应倾向性。 一般来说，强电解质的离子可以共存，而某些离子可能因生成沉淀或气体而不能共存。 例如，Ag⁺与Cl⁻不能共存，因为它们会生成AgCl沉淀；而Na⁺与NO3⁻可以共存，因为它们形成的盐均可溶于水。

1.3 氧化还原反应
  1. 氧化还原反应的定义

    氧化还原反应是指在化学反应中发生电子转移的过程，其中一种物质失去电子（氧化），另一种物质获得电子（还原）。 氧化剂是能够接受电子的物质，而还原剂是能够提供电子的物质。 例如，在Zn与Cu²⁺反应中，Zn失去电子被氧化为Zn²⁺，Cu²⁺获得电子被还原为Cu。

  2. 氧化剂与还原剂的识别

    识别氧化剂和还原剂的方法是观察反应中物质的化合价变化。 失去电子（化合价升高）的物质是还原剂，获得电子（化合价降低）的物质是氧化剂。 例如，在Fe + Cu²⁺ → Fe²⁺ + Cu反应中，Fe被氧化为Fe²⁺（还原剂），Cu²⁺被还原为Cu（氧化剂）。 rule: 在配平氧化还原反应方程式时，首先确定各物质的化合价变化，然后根据电子守恒原则进行配平。

  3. 常见氧化还原反应实例

    常见的氧化还原反应实例包括金属与酸反应生成氢气（如Zn + 2HCl → ZnCl2 + H2）、金属与氧气反应生成氧化物（如4Fe + 3O2 → 2Fe2O3）以及置换反应（如Cu + 2AgNO3 → Cu(NO3)2 + 2Ag）。 其他的氧化还原反应还包括燃烧反应（如CH4 + 2O2 → CO2 + 2H2O）和电化学反应（如电池中的反应）。 这些反应中均涉及电子的转移，体现了氧化还原反应的本质。

  4. 化合价变化与电子转移

    化合价变化是判断氧化还原反应的关键，通过比较反应前后各元素的化合价来确定电子的转移情况。 失去电子导致化合价升高的物质被氧化，获得电子导致化合价降低的物质被还原。 例如，在反应2Fe + 3Cl2 → 2FeCl3中，Fe的化合价从0变为+3（失去3个电子，被氧化），Cl的化合价从0变为-1（每个Cl获得1个电子，被还原）。 再如歧化反应，2H2O2 → 2H2O + O2中，氧元素在H2O2中化合价为-1，在H2O中为-2（还原），在O2中为0（氧化）。

  5. 氧化还原方程式配平（电子守恒）

    配平氧化还原方程式时，需遵循电子守恒原则，即反应中失去的电子数等于获得的电子数。 常用的方法包括离子电子法和氧化数变化法，步骤包括确定氧化剂和还原剂、写出半反应方程式、配平电子数以及合并半反应方程式。 例如，在Fe + Cu²⁺ → Fe²⁺ + Cu反应中，Fe的氧化半反应为Fe → Fe²⁺ + 2e⁻，Cu的还原半反应为Cu²⁺ + 2e⁻ → Cu，合并后得到配平后的方程式。 再如，MnO4⁻ + Fe²⁺ → Mn²⁺ + Fe³⁺反应中，需分别配平Mn和Fe的半反应，然后合并配平电子数，最终得到完整的氧化还原方程式， 步骤如下：1. 写出氧化和还原半反应；MnO4⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H2O 和 Fe²⁺ → Fe³⁺ + e⁻； 2. 配平各半反应中的元素和电荷； 3. 使电子数相等后合并半反应； 4. 检查并确保方程式质量守恒和电荷守恒。

第二章：海水中的重要元素——钠和氯
2.1 钠及其化合物
  1. 金属钠的物理性质与保存

    金属钠为银白色金属，质软易切割，密度小于水，能浮在水面上。熔点为97.8℃，沸点为883℃。 钠具有良好的导电性和导热性，化学性质活泼，易与空气中的氧气和水反应。 由于钠易与空气中的氧气和水分反应生成氧化钠和氢氧化钠，因此通常将其保存在石蜡油或煤油中以防止氧化和水解。

  2. 钠与水、氧的反应

    钠与水反应，生成氢氧化钠和氢气，反应剧烈并放出大量热量，易引发燃烧。“浮游溶响红” 现象是钠与水反应时，钠块在水面上快速移动并发出红色火焰的现象。 钠与氧气反应，常温下生成氧化钠（Na2O），点燃则主要生成过氧化钠（Na2O2），反应剧烈并放出热量。

  3. Na₂O 与 Na₂O₂ 的性质区别

    氧化钠（Na2O）为白色固体，易溶于水生成强碱性溶液氢氧化钠（NaOH），具有强碱性。 过氧化钠（Na2O2）为淡黄色固体，也易溶于水，生成氢氧化钠和过氧化氢（H2O2），具有氧化性。 Na2O与Na2O2在化学性质上有明显区别，前者主要表现为碱性，后者则同时具有较强的氧化性。

  4. Na₂CO₃ 与 NaHCO₃ 的性质对比

    碳酸钠（Na2CO3）为白色结晶性固体，易溶于水，溶液呈碱性，常用于制造玻璃和洗涤剂。 碳酸氢钠（NaHCO3）为白色结晶性固体，也易溶于水，溶液呈弱碱性，常用于食品添加剂和医药制剂。 Na2CO3与NaHCO3在化学性质上有所不同，前者为强碱性盐，后者为弱碱性盐，且NaHCO3在加热时会分解生成Na2CO3、CO2和H2O。 如何只用盐酸鉴别两者：向少量固体样品中加入稀盐酸，NaHCO3会立即产生大量气泡（CO2），而Na2CO3反应较慢且气泡较少。

  5. 焰色反应与钠的应用

    焰色反应是指某些金属元素在火焰中燃烧时产生特定颜色的现象。常用于鉴别金属离子。 钠的焰色反应为黄色，常用于化学分析中鉴别钠离子。 钠及其化合物广泛应用于工业和日常生活中，如金属钠用于有机合成和还原剂，碳酸钠用于制造玻璃、肥皂和洗涤剂，氢氧化钠用于造纸、纺织和水处理等领域。

2.2 氯及其化合物
  1. 氯气的性质与制备

    氯气（Cl2）为黄绿色气体，有刺激性气味，密度较空气大，不易溶于水但易溶于有机溶剂。有极强的氧化性。 氯气具有强氧化性和消毒作用，能与多种物质反应生成氯化物。 实验室制备氯气常用的方法是通过浓盐酸与二氧化锰共热反应（MnO2 + 4HCl → MnCl2 + 2H2O + Cl2↑）或电解氯化钠溶液（NaCl电解）。

  2. 氯水的成分与漂白机理

    氯水是氯气溶解在水中形成的溶液，主要成分包括Cl2、HCl和HClO（次氯酸）。 漂白机理主要是由于次氯酸（HClO）的强氧化性，能够破坏有色物质的分子结构，使其失去颜色，从而达到漂白效果。 氯水在消毒和漂白中广泛应用，但使用时需注意其刺激性和腐蚀性。

  3. 氯气与各类物质的反应

    氯气能与多种物质发生反应，包括与金属反应生成氯化物（如2Na + Cl2 → 2NaCl）、与非金属反应生成卤化物（如H2 + Cl2 → 2HCl）、与水反应生成氯水（Cl2 + H2O → HCl + HClO）等。 氯气的强氧化性使其能够与许多有机和无机物质反应，生成各种氯化合物。 这些反应在工业生产和实验室中具有重要应用价值。 比如漂白粉的制备：2Ca(OH)2 + 2Cl2 → Ca(ClO)2 + CaCl2 + 2H2O 漂白液的制备：Cl2 + 2NaOH → NaCl + NaClO + H2O

    4. 氯气的实验室与工业制法

    实验室制备氯气常用的方法是通过浓盐酸与二氧化锰共热反应（MnO2 + 4HCl → MnCl2 + 2H2O + Cl2↑）或电解氯化钠溶液（NaCl电解）。 这些方法能够有效生成氯气，适用于实验室规模的制备。 工业上，氯气主要通过电解饱和氯化钠溶液（盐水电解）制备，产生氯气、氢气和氢氧化钠。 该方法具有高效、经济的优点，广泛应用于工业生产中。

  4. 氯离子的检验方法

    氯离子的检验常用的方法是加入硝酸银溶液（AgNO3），生成白色沉淀氯化银（AgCl）。 具体步骤为：取待测溶液，加入几滴硝酸银溶液，若生成白色沉淀，则说明溶液中含有氯离子。 该方法简单、直观，是实验室中常用的氯离子检验方法。

  5. 含氯消毒与安全防护

    含氯消毒剂（如氯水、漂白粉）广泛用于饮用水消毒、环境卫生和医疗器械消毒，具有杀菌灭病毒的作用。 使用含氯消毒剂时需注意安全防护，避免吸入氯气或接触皮肤和眼睛，以防引起刺激或伤害。 在使用过程中，应在通风良好的环境下操作，佩戴适当的防护装备，如手套和护目镜。

2.3 物质的量
  1. 物质的量与阿伏伽德罗常数

    物质的量（n）是表示物质中微粒数量的物理量，单位为摩尔（mol）。1 mol物质含有6.022×10²³个微粒（原子、分子、离子等），该数值称为阿伏伽德罗常数（NA）。 物质的量是化学计算中的基本概念，广泛应用于化学反应方程式的配平和定量分析中。

  2. n = m/M 与 n = V/Vm 的应用

    物质的量的计算常用公式包括n = m/M（m为物质的质量，M为物质的摩尔质量）和n = V/Vm（V为气体体积，Vm为气体的摩尔体积，标准状况下Vm约为22.4 L/mol）。 这些公式用于不同类型物质的物质的量计算，帮助解决实际问题。 例如，计算一定质量的物质含有多少摩尔，或在标准状况下某体积气体含有多少摩尔。 big tip: 标准状况下气体的摩尔体积为22.4 L/mol，常温常压下约为24 L/mol。

  3. 溶液物质的量浓度（c = n/V）计算

    物质的量浓度（c）表示溶液中溶质的物质的量与溶液体积的比值，单位为mol/L。计算公式为c = n/V，其中n为溶质的物质的量，V为溶液体积。 该概念用于描述溶液的浓度，广泛应用于化学实验和工业生产中。 例如，计算一定体积溶液中溶质的物质的量，或根据已知物质的量浓度配制溶液。

  4. 气体体积与物质的量关系

    在标准状况下，气体的体积与其物质的量成正比关系，1 mol气体的体积约为22.4 L。 该关系用于计算气体的物质的量或体积，帮助解决涉及气体的化学问题。 例如，已知气体体积可计算其物质的量，反之亦然。 遵循公式PV = nRT，其中P为压力，V为体积，n为物质的量，R为气体常数，T为温度（开尔文）。

实验活动 1：配制一定物质的量浓度的溶液
  1. 容量瓶使用规范

    容量瓶是细颈、梨形的平底玻璃瓶，瓶口配有磨口玻璃塞或塑料塞。 容量瓶常用于配制一定体积、一定浓度的溶液。容量瓶上标有温度（一般为20°C） 和容积，表示在所指温度下，液体的凹液面与容量瓶颈部的刻度线相切时，溶液体积 恰好与瓶上标注的容积相等。使用容量瓶时要注意以下几点：（1）容量瓶瓶塞须用 结实的细绳系在瓶颈上，以防止损坏或丢失。（2）在使用前，首先要检查容量瓶是否 完好，瓶口处是否漏水。经检查不漏水的容量瓶才能使用。（3）容量瓶使用完毕， 应洗净、晾干。（对于玻璃磨口瓶塞，应在瓶塞与瓶口处垫一张纸条，以免瓶塞与 瓶口粘连。

  2. 定容与读数要点

    配制溶液时，先将溶质溶解在少量溶剂中，待溶质完全溶解后，将溶液倒入容量瓶中。 然后用溶剂定容至刻度线。定容时，应注意以下几点：（1）定容时应慢慢加入溶剂， 以免超过刻度线。（2）定容至刻度线时，应使液面与刻度线相切，视线应与刻度线 保持水平。（3）定容后，应将容量瓶塞紧，轻轻摇动容量瓶，使溶液混合均匀。

第三章：铁与金属材料
3.1 铁及其化合物
  1. 铁的性质与常见氧化态

    铁在自然界中可以像陨铁中的铁那样以单质形态存在，但主要是以+2 价和+3 价化合物的形态存在于矿石中。铁元素在地壳中的含量仅次于氧、硅和铝，居第四位。

  2. Fe²⁺/Fe³⁺的相互转化

    Fe²⁺和Fe³⁺是铁的两种常见氧化态，它们可以通过氧化还原反应相互转化。Fe²⁺可以被氧化剂氧化为Fe³⁺，而Fe³⁺可以被还原剂还原为Fe²⁺。这种转化在铁的化学性质和应用中具有重要意义。 比如，用氧气或过氧化氢可以将Fe²⁺氧化为Fe³⁺，而用亚硫酸钠或铁粉可以将Fe³⁺还原为Fe²⁺，这两个反应的离子方程式如下： O2 + 4H⁺ + 4Fe²⁺ → 2H2O + 4Fe³⁺ Fe³⁺ +SO2 + 2H2O → Fe²⁺ + SO4²⁻ + 4H⁺

  3. 铁盐的检验与应用

    铁盐主要包括Fe²⁺盐和Fe³⁺盐，它们在化学实验和工业生产中有广泛的应用。铁盐的检验方法主要是通过与氰化钾（K₃[Fe(CN)₆]）反应生成不同颜色的沉淀来区分Fe²⁺和Fe³⁺。Fe²⁺与K₃[Fe(CN)₆]反应生成蓝色沉淀（普鲁士蓝），而Fe³⁺与K₄[Fe(CN)₆]反应生成棕色沉淀。 另外，氢氧化钠在高浓度下也可以用来区分Fe²⁺和Fe³⁺，Fe²⁺生成白色沉淀后迅速变为灰绿色，最后变为红棕色，而Fe³⁺生成棕色沉淀（Fe(OH)₃）。 铁盐在水处理、颜料制造和医药等领域有重要应用。例如，FeCl3常用于净化饮用水，FeSO4用于治疗贫血。

3.2 金属材料
  1. 合金的组成与性质

    合金是由两种或多种金属元素组成的固体溶液，具有优于单一金属的机械性能和化学性质。常见的合金包括钢（铁与碳的合金）、不锈钢（含铬、镍等元素的钢）、黄铜（铜与锌的合金）等。合金的性质取决于其组成和微观结构，通常具有更高的强度、硬度和耐腐蚀性。

  2. 钢与不锈钢的应用

    钢是一种重要的工程材料，广泛应用于建筑、机械制造、交通运输等领域。根据碳含量的不同，钢可以分为低碳钢、中碳钢和高碳钢，不同类型的钢具有不同的机械性能和用途。不锈钢由于其优异的耐腐蚀性，常用于厨房用具、医疗器械和化工设备等领域。

  3. 金属腐蚀与防护

    金属腐蚀是指金属与环境中的物质发生化学反应，导致金属表面损坏和性能下降的现象。常见的腐蚀形式包括氧化腐蚀、电化学腐蚀等。防护方法主要包括涂层保护（如油漆、镀锌）、阴极保护和合金化等。这些方法能够有效延长金属材料的使用寿命，减少经济损失。

第四章：物质结构与元素周期律
4.1 原子结构与元素周期表
  1. 核外电子排布与周期表

    原子结构包括原子核和核外电子，核外电子按照能级和亚能级进行排布。元素周期表根据原子序数排列元素，反映了元素的周期性性质。电子排布规律遵循能量最低原则、泡利不相容原理和洪特规则。 元素周期表分为周期和族，周期表示电子层数，族表示元素的化学性质相似性。通过了解原子结构和电子排布，可以解释元素的化学性质和周期律。

  2. 周期与族的意义

    周期表示元素的电子层数，随着周期数的增加，元素的原子半径增大，化学性质逐渐变化。族表示元素的化学性质相似性，同族元素具有相似的电子排布和化学性质。 例如，碱金属族（第1族）元素具有相似的化学性质，如与水反应生成氢气和氢氧化物。通过理解周期和族的意义，可以预测元素的化学行为和反应趋势。

  3. 碱金属元素

    碱金属元素包括锂（Li）、钠（Na）、钾（K）、铷（Rb）、铯（Cs）和钫（Fr），它们位于元素周期表的第1族。碱金属具有较低的电离能和较大的原子半径，化学性质非常活泼，易与水反应生成氢气和氢氧化物。 碱金属在自然界中主要以化合物形式存在，广泛应用于工业和日常生活中，如钠用于制造玻璃和肥皂，钾用于农业肥料等。

  4. 卤族元素

    卤族元素包括氟（F）、氯（Cl）、溴（Br）、碘（I）和砹（At），它们位于元素周期表的第17族。卤族元素具有较高的电负性和较小的原子半径，化学性质活泼，易与金属反应生成卤化物。 卤族元素在自然界中主要以化合物形式存在，广泛应用于消毒、漂白和医药等领域，如氯用于水处理和消毒，碘用于防治甲状腺疾病等。

4.2 元素周期律
  1. 元素性质的周期性变化

    元素的性质随着原子序数的增加呈现周期性变化，这种现象称为元素周期律。周期律反映在原子半径、电离能、电子亲和能和电负性等性质上。 例如，随着周期的增加，原子半径逐渐增大，而电离能和电负性则呈现出周期性的变化趋势。通过理解元素周期律，可以解释元素的化学性质和反应行为。

  2. 同周期、同主族的性质递变

    同周期元素的性质递变主要体现在原子半径、电离能和电负性等方面。随着原子序数的增加，原子半径逐渐减小，而电离能和电负性则逐渐增大。 同主族元素的性质递变则表现为化学性质的相似性和物理性质的变化。例如，碱金属族元素（第1族）具有相似的化学性质，但随着原子序数的增加，其密度、熔点和沸点等物理性质逐渐变化。

4.3 化学键
  1. 化合物的电子式

    化合物的电子式表示化合物中原子间的电子分布情况，反映了化学键的形成和性质。常见的电子式包括路易斯结构式和共价键电子式。 通过绘制化合物的电子式，可以了解原子间的键合方式、键的类型以及分子的几何结构，从而解释化合物的化学性质和反应行为。

  2. 离子键与共价键

    离子键是由正负离子之间的静电引力形成的化学键，通常发生在金属与非金属元素之间。共价键是由原子间共享电子对形成的化学键，通常发生在非金属元素之间。 离子键和共价键具有不同的性质，如熔点、沸点、电导性等。通过理解离子键和共价键的形成机制，可以解释化合物的结构和性质。

  3. 化学键与物质性质关系

    化学键的类型和强度直接影响物质的物理和化学性质。离子键形成的化合物通常具有高熔点和沸点，良好的电导性，而共价键形成的化合物则表现出较低的熔点和沸点，绝缘性较好。 通过分析化学键与物质性质的关系，可以预测和解释不同化合物的行为和应用。
